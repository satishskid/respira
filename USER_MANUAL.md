## Troubleshooting

### "I can't hear the AI voice"

**Check:**
1. Volume is up on your device
2. Microphone permission granted
3. Your Gemini API key is entered correctly
4. Browser supports Web Speech API (use Chrome/Edge)
5. Try refreshing the page

### "The AI isn't responding to my voice"

**Check:**
1. Microphone permission granted
2. Speak clearly and at normal volume
3. Wait for coach to finish speaking before interrupting
4. Check if microphone icon shows active (usually green)

### "Session won't start"

**Possible Causes:**
1. **No API Key:** Go to Settings ‚Üí Add your Gemini API key
2. **Invalid API Key:** Keys start with "AIza" - check for typos
3. **No Microphone:** Grant microphone access when prompted
4. **Browser Compatibility:** Switch to Chrome or Edge

### "Rain/ambient sound but no voice"

**This means:**
- Ambient audio is playing (background sound)
- But AI session hasn't started yet

**Solution:**
1. Click the "Start" button
2. Make sure you completed onboarding
3. Check that API key is set in Settings

### "API Key Error"

**Common Issues:**
1. **Wrong Format:** Key must start with "AIza"
2. **Expired Key:** Generate new key at Google AI Studio
3. **API Not Enabled:** Enable Gemini API in Google Cloud Console
4. **Rate Limit:** Wait a few minutes and try again

### "Session cuts out mid-way"

**Possible Causes:**
1. **Network Issue:** Check internet connection
2. **API Quota:** You may have hit daily free limit (~50 requests/day)
3. **Browser Tab Inactive:** Keep tab in foreground during session

## FAQ

### General

**Q: Is RESPIRA free?**  
A: The app is free. You pay Google ~$0.01 per session for API usage. No subscription. No hidden fees.

**Q: Do I need an account?**  
A: No. You can use as guest. Google sign-in is optional (helps sync across devices).

**Q: Can I use offline?**  
A: No. You need internet to connect to Google's Gemini API for the AI voice.

**Q: Does it work on iPhone?**  
A: Yes! Use Safari or Chrome. Install as PWA for app-like experience.

**Q: What about Android?**  
A: Yes! Chrome recommended. Install as PWA.

### Privacy

**Q: Is my voice recorded?**  
A: No. Your voice is processed in real-time by Google's API and immediately discarded. Never stored.

**Q: Can you see my API key?**  
A: No. It's stored locally in your browser. We have no servers to see it.

**Q: What data do you collect?**  
A: Zero. Everything is local. We don't have analytics, tracking, or cloud storage.

**Q: Can I delete my data?**  
A: Yes. Settings ‚Üí Clear All Data. Or clear browser data.

### Technical

**Q: Which browsers work best?**  
A: Chrome and Edge (best). Safari works but some features limited. Firefox not recommended.

**Q: Why do I need a Gemini API key?**  
A: RESPIRA runs on your device (zero-knowledge architecture). Your key connects your browser directly to Google's AI‚Äîno middleman.

**Q: How do I get an API key?**  
A: Visit [aistudio.google.com/apikey](https://aistudio.google.com/apikey) ‚Üí Sign in ‚Üí Create API Key ‚Üí Copy it ‚Üí Paste in RESPIRA Settings.

**Q: What if I hit API limits?**  
A: Google gives 50 free requests/day. Each session = ~1-3 requests. Upgrade to paid tier if needed.

**Q: Can I use my own AI model?**  
A: Not yet. Currently only supports Google Gemini Live API.

### Usage

**Q: How long are sessions?**  
A: 3-15 minutes depending on protocol. You can stop anytime.

**Q: Can I skip protocols?**  
A: Yes! Use whichever protocol fits your need. Circadian timing is optimal, not required.

**Q: What if I have a health condition?**  
A: Tell RESPIRA during onboarding. The AI will automatically avoid unsafe techniques.

**Q: Can I use during pregnancy?**  
A: Yes, but enable "Pregnancy" in safety settings. RESPIRA will skip breath retention and forceful breathing.

**Q: Is this a replacement for therapy?**  
A: No. RESPIRA is a breathwork tool for stress management and focus. Not medical treatment.
